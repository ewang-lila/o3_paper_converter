Problem 1

Background. In ferroelectric materials the dielectric permittivity ( \varepsilon' ) above the phase transition temperature can be described by the Curie-Weiss law
[ \frac{1}{\varepsilon'} = \frac{T-T_0}{C}, ] where ( T_0 ) is the Curie-Weiss temperature and ( C ) is the Curie-Weiss constant. In relaxor ferroelectrics, the dielectric maximum occurs at a frequency-dependent temperature ( T_M ) with peak value ( \varepsilon_M'(f) ). To phenomenologically capture the diffuse phase transition, the Curie-Weiss law is modified by introducing a diffuseness exponent ( \gamma ) so that the deviation from the maximum reciprocal permittivity is given as a power law in ( T-T_M ).

Task. Derive the modified expression for the dielectric response that defines the diffuseness parameter ( \gamma ) in terms of ( \varepsilon'(T) ), ( \varepsilon_M'(f) ), ( T-T_M(f) ), and ( C ).

Solution:
[ \frac{1}{\varepsilon'}-\frac{1}{\varepsilon_M'(f)}=\frac{1}{C}\,(T-T_M(f))^\gamma. ] (Equation (2) in the paper)

Problem 2

Background. In perovskite ferroelectric materials such as lead scandium tantalate (PST), the degree of structural B‐site ordering is characterized by comparing the intensities of two x-ray diffraction peaks. The intensity ( I_{111} ) corresponds to the pseudocubic (111) superstructure peak that appears only when there is ordering, and ( I_{200} ) corresponds to the symmetry-allowed fundamental (200) peak. For a perfectly ordered system, the theoretical intensity ratio is denoted by
[ \left(\frac{I_{111}}{I_{200}}\right)_{S=1}. ]
An ordering coefficient ( S ) is introduced such that its square scales the measured intensity ratio to the ideal one.

Task. Derive the expression for the ordering coefficient ( S ) in terms of the measured intensity ratio ( I_{111}/I_{200} ) and the theoretical ratio for a perfectly ordered system.

Solution:
[ S^2 = \frac{\left(\frac{I_{111}}{I_{200}}\right)}{\left(\frac{I_{111}}{I_{200}}\right)_{S=1}}. ] (Equation (3) in the paper)

Problem 3

Background. Experimental studies on PST have revealed that the diffuseness of the phase transition, quantified by the exponent ( \gamma ), depends on the degree of structural ordering represented by the coefficient ( S ). When the sample is fully disordered (( S=0 )), relaxor behavior is manifested with ( \gamma ) near 2, whereas nearly perfect ordering (( S ) close to 1) yields ferroelectric behavior with ( \gamma ) around 1. A linear relation between ( \gamma ) and ( S ) has been found to describe the experimental data across different sample geometries.

Task. Derive the linear relation between the diffuseness exponent ( \gamma ) and the ordering parameter ( S ).

Solution:
[ \gamma(S) = 1.93 - 1.06\,S. ] (As stated in the paper)

Problem 1

Background.
Consider the variance‐preserving (VP) stochastic differential equation (SDE) given by
[ \d{x} = -\frac{1}{2}\beta(t)\, x\,\d{t} + \sqrt{\beta(t)}\,\d{w}, ]
with the time‐dependent noise scale defined by
[ \beta(t) = \beta_{\min} + t\,(\beta_{\max}-\beta_{\min}). ]
For such an affine SDE, the transition probability from the initial condition ( x(0) ) to the state ( x(t) ) can be written in closed form as a Gaussian distribution.

Task.
Show that the transition kernel for this VP diffusion SDE can be expressed as
[ p_t(x(t) \mid x(0)) = \mathcal{N}!\Bigl(x(t);\, \mu(x(0),t),\, \sigma(t)\,I\Bigr), ]
with the mean ( \mu(x(0),t) ) and variance ( \sigma(t) ) given in terms of the cumulative noise function. That is, derive the expression for
[ \mu(x(0),t)=\exp!\left(-\tfrac{1}{2}\int_0^t\beta(s)\,\d{s}\right)x(0) ]
and
[ \sigma(t)=1-\exp!\left(-\int_0^t\beta(s)\,\d{s}\right). ]

Solution:
The final result appears verbatim in the paper as
[ p_t(x(t) \mid x(0)) = \mathcal{N}\Bigl(x(t);\, e^{-\frac{1}{2}\int_{0}^{t} \beta(s)\,\d{s}}\; x(0),\, \bigl(1 - e^{-\int_{0}^{t} \beta(s)\,\d{s}}\bigr)\,I\Bigr). ] ——— (see Section “Diffusion” in the paper)

Problem 3

Background.
For a diffusion process governed by
[ \d{x} = f(x,t)\,\d{t} + g(t)\,\d{w}, ]
the probability density ( p_t(x) ) evolves in time according to the Fokker–Planck equation. In logarithmic form, this equation relates the time-derivative of the log-density to derivatives of ( \log p_t(x) ) and the drift ( f(x,t) ).

Task.
Show that the log-density of ( p_t(x) ) satisfies the following equation: [ \partial_t \log p_t(x) = \frac{1}{2}\,g^2(t)\left[\operatorname{div}_x!\bigl(\nabla_x \log p_t(x)\bigr) + \left\Vert \nabla_x \log p_t(x) \right\Vert_2^2\right] - \langle f(x,t),\,\nabla_x \log p_t(x) \rangle - \operatorname{div}_x\bigl(f(x,t)\bigr). ]

Solution:
The paper states the Fokker–Planck equation for the log-density verbatim as
[ \partial_t \log p_t(x) = \frac{1}{2}\,g^2(t)\left[\operatorname{div}_x!\bigl(\nabla_x \log p_t(x)\bigr) + \left\Vert \nabla_x \log p_t(x) \right\Vert_2^2\right] - \langle f(x,t),\,\nabla_x \log p_t(x) \rangle - \operatorname{div}_x\bigl(f(x,t)\bigr). ] ——— (see Equation (Fokker-Planck) in the paper)

Problem 4

Background.
To enforce that an energy-based diffusion model satisfies the Fokker–Planck equation, the paper introduces a residual
[ R(x,t) = \mathcal{F}_{p}(x,t) - \partial_t \log p_t(x), ]
where
[ \mathcal{F}_{p}(x,t)= \frac{1}{2}\,g^2(t)\left[\operatorname{div}_x\bigl(\nabla_x \log p_t(x)\bigr) + \|\nabla_x \log p_t(x)\|_2^2\right] - \langle f(x,t), \nabla_x \log p_t(x)\rangle - \operatorname{div}_x\bigl(f(x,t)\bigr). ]
Since directly computing higher-order derivatives (e.g. divergence) is computationally costly, a weak residual formulation is introduced. For a small parameter (\sigma>0) and a perturbation ( v \sim \mathcal{N}(0,\sigma^2 I) ), one defines an unbiased estimator ( \widetilde{R}(x,t;v) ) of the residual using only first-order derivative information.

Task.
Show that an unbiased estimator of the residual can be written as
[ \widetilde{R}(x,t; v) = \frac{1}{2}\,g^2(t)\,\left[ \left(\frac{v}{\sigma}\right)^\top \frac{s(x+v,t)-s(x-v,t)}{2\sigma} + \|s(x+v,t)\|_2^2\right] - \langle f(x+v,t),\, s(x+v,t)\rangle - \operatorname{div}_x\bigl(f(x+v,t)\bigr) - \partial_t\log p_t(x+v), ] where the score ( s(x,t) = \nabla_x \log p_t(x) ).

Solution:
The final result in the paper is given verbatim as
[ \widetilde{R}(x,t; v) = \frac{1}{2}\,g^2(t)\,\left[ \left(\frac{v}{\sigma}\right)^\top \frac{s(x+v,t)-s(x-v,t)}{2\sigma} + \|s(x+v,t)\|_2^2\right] - \langle f(x+v,t),\, s(x+v,t)\rangle - \operatorname{div}_x\bigl(f(x+v,t)\bigr) - \partial_t\log p_t(x+v). ] This yields an unbiased estimator for the weak residual, which can then be used within the Fokker–Planck regularization loss.
——— (see Theorem “Weak Residual Formulation” in the paper)

Problem 1

Background. Consider a two‐dimensional system whose local order parameters near a corner exhibit a power‐law behavior characterized by critical exponents. Let $\beta_b$ be the boundary (edge) magnetization critical exponent and $\beta_c$ be the corner magnetization critical exponent. The angle between the two adjacent edges forming the corner is denoted by $\theta$. In the framework of boundary conformal field theory and as shown by Cardy, there is a simple relation connecting these two exponents.

Task. Derive an analytic expression for the corner critical exponent $\beta_c$ in terms of the boundary exponent $\beta_b$ and the angle $\theta$.

Solution:
[ \beta_c = \frac{\pi}{\theta}\,\beta_b. ] (Quoted verbatim from the paper: “The relation of these two critical exponents is
[ \beta_c = \frac\pi\theta\beta_b, ] where $\theta$ is the angle between the edges of the corner.”)

Problem 2

Background. The two-dimensional Ising model on a square lattice is described by the Hamiltonian
[ H = -\sum_{\langle i,j\rangle} \sigma_i\sigma_j, ] with spins $\sigma_k\in{0,1}$ and the summation running over nearest neighbor pairs. The exact analytical result for the magnetization $M$ in the bulk (i.e. measured at the center of the system) was derived by Yang. The expression involves the hyperbolic sine function and is expressed in reduced units where temperature enters as $1/T$.

Task. Derive the analytic expression for the bulk magnetization $M$ as a function of the temperature $T$.

Solution:
[ M = \left[ 1- \frac{1}{\sinh^4\left(\frac{1}{T}\right)}\right]^{\frac{1}{8}}. ] (Quoted verbatim from the paper, Equation (exactbulk):
“\begin{eqnarray}\label{exactbulk} M &= \left[ 1- \frac1{\sinh^4\left(\frac1T\right)}\right]^{\frac18},\ \end{eqnarray}”)

Problem 3

Background. For the two-dimensional Ising model, the magnetization measured at the boundary (specifically, on the middle of the border) is given by an exact analytic formula. This expression accounts for the reduced coordination at the edge and involves hyperbolic functions evaluated at $2/T$. The boundary magnetization, denoted by $M_b$, provides a distinct critical behavior compared to the bulk.

Task. Derive the analytic expression for the boundary magnetization $M_b$ in terms of the temperature $T$.

Solution:
[ M_b = \sqrt{\frac{\cosh\left(\frac{2}{T}\right) - \coth\left(\frac{2}{T}\right)}{\cosh\left(\frac{2}{T}\right)-1}}. ] *(Quoted verbatim from the paper, Equation (\ref{exactcorner}):
“\begin{eqnarray}\label{exactcorner} M_b &= \sqrt{\frac{\cosh\left(\frac2T\right) - \coth\left(\frac2T\right)}{\cosh\left(\frac2T\right)-1}},\ \end{eqnarray}”)*

Problem 4

Background. In order to extract the critical exponent $\beta$ from numerical data, one considers the ratio
[ R(T) = \frac{M}{M'}, ] where $M$ is the magnetization and $M'=\frac{dM}{dT}$ its first derivative with respect to temperature. Near the critical temperature $T_C$, the non-analytical part of the temperature dependence can be removed by a Taylor expansion of $R(T)$ in powers of $(T_C-T)$. This leads to a fitting function characterized by a linear term with coefficient $-1/\beta$ and a quadratic correction with coefficient $c$.

Task. Derive the analytic form of the fitting function for the ratio $R(T)$ as a function of $T$, including the critical exponent $\beta$, the critical temperature $T_C$, and the coefficient $c$.

Solution:
[ R(T) = -\frac{1}{\beta}\left(T_C - T\right) + c\left(T_C - T\right)^2. ] (Quoted verbatim from the paper, Equation (\ref{rfit}):
“\begin{equation}\label{rfit} R(T) = -\frac1\beta\left(T_C - T\right) + c\left(T_C - T\right)^2, \end{equation}”)
